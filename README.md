# ⚽ Bruninho e Simões — Sistema de Notas do Flamengo

Sistema completo para o canal [Bruninho e Simões](https://www.youtube.com/@BruninhoeSimoes) dar notas aos jogadores do Flamengo após cada partida.

## Stack
- **Frontend:** Next.js 14 → Vercel
- **Backend:** Node.js + Express → Render
- **Banco:** PostgreSQL → Supabase
- **Auth:** JWT + bcrypt

## Início Rápido

Leia o guia completo: **[DEPLOY.md](./DEPLOY.md)**
